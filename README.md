# Little guide how to use
Firstly you need to type. Sudo xbps-install -S . to install xbps. \n
Then sudo xbps-install -S git \n
And now git clone https://github.com/Chaos4U/dddats/ \n
cd dddats/ \n
chmod +x star.sh \n
./star.sh \n
